What is Zot Crypto?

Zot crypto is an open source easy to use end-to-end encryption dependency for multiple languages.

It aims to develop a dependency for small developers who can use the dependency to establish basic level of encryption.

We aim to develop reliable dependencies for RSA, AES and x509 certificates.
